Estructura Básica de Flexbox

Para activar Flexbox, se debe aplicar display: flex; a un contenedor. Esto hace que sus hijos se distribuyan en una línea o columna flexible.

Dirección del Eje Principal

La propiedad flex-direction define la dirección en la que los elementos flexibles se organizan dentro del contenedor.

Fila (row)
Columna (column)
Fila inversa (row-reverse)
Columna inversa (column-reverse)

Valores posibles:

  • row: Dirección predeterminada, los elementos se organizan en una fila.
  • column: Los elementos se organizan en una columna.
  • row-reverse: Los elementos se organizan en fila, pero en orden inverso.
  • column-reverse: Los elementos se organizan en columna, pero en orden inverso.

Distribución de Elementos en el Eje Principal

La propiedad justify-content controla la alineación de los elementos en el eje principal del contenedor.

Inicio (flex-start)
Centro (center)
Final (flex-end)
Espaciado entre (space-between)
Espaciado alrededor (space-around)
Espaciado uniforme (space-evenly)

Valores posibles:

  • flex-start: Los elementos se alinean al inicio del contenedor.
  • center: Los elementos se alinean al centro del contenedor.
  • flex-end: Los elementos se alinean al final del contenedor.
  • space-between: Los elementos se distribuyen con el mayor espacio posible entre ellos.
  • space-around: Los elementos tienen espacio alrededor de ellos.
  • space-evenly: El espacio entre todos los elementos es uniforme.

Alineación de Elementos en el Eje Secundario

La propiedad align-items controla la alineación de los elementos en el eje secundario del contenedor.

Estirar (stretch)
Inicio (flex-start)
Centro (center)
Final (flex-end)
Línea base (baseline)

Valores posibles:

  • stretch: Los elementos se estiran para ocupar toda la altura del contenedor.
  • flex-start: Los elementos se alinean al inicio del eje secundario.
  • center: Los elementos se alinean al centro del eje secundario.
  • flex-end: Los elementos se alinean al final del eje secundario.
  • baseline: Los elementos se alinean según su línea base de texto.

Distribución de Múltiples Líneas

La propiedad align-content controla la distribución del espacio entre líneas cuando hay varias en un contenedor flexible con flex-wrap.

Estirar (stretch)
Inicio (flex-start)
Centro (center)
Final (flex-end)
Espaciado entre (space-between)
Espaciado alrededor (space-around)

Valores posibles:

  • stretch: Las líneas se estiran para ocupar todo el espacio disponible.
  • flex-start: Las líneas se alinean al inicio del contenedor.
  • center: Las líneas se alinean al centro del contenedor.
  • flex-end: Las líneas se alinean al final del contenedor.
  • space-between: Las líneas se distribuyen con el mayor espacio posible entre ellas.
  • space-around: Las líneas tienen espacio alrededor de ellas.

Propiedades Individuales de los Elementos Flexibles

Los elementos flexibles dentro de un contenedor pueden comportarse de manera independiente con propiedades como order, flex-grow, flex-shrink, flex-basis y align-self.

Orden (order)
Crecimiento (flex-grow)
Encogimiento (flex-shrink)
Tamaño Base (flex-basis)
Alineación Individual (align-self)

Explicación de las propiedades:

  • order: Define el orden en que se muestran los elementos dentro del contenedor.
  • flex-grow: Controla cuánto puede crecer un elemento en comparación con otros.
  • flex-shrink: Determina cuánto puede reducirse un elemento si no hay suficiente espacio.
  • flex-basis: Especifica el tamaño inicial de un elemento antes de aplicar flex-grow o flex-shrink.
  • align-self: Permite alinear un solo elemento de manera diferente al resto en el eje secundario.
