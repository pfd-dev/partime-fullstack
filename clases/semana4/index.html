JavaScript

JavaScript es un lenguaje de programación que permite agregar interactividad y dinamismo a las páginas web. Se ejecuta en el navegador y es utilizado para crear efectos visuales, validar formularios, manipular elementos HTML y mucho más.

¿Cómo ejecutar JavaScript?

JavaScript se puede ejecutar en una página web de tres maneras:

  • En línea: Dentro de etiquetas HTML usando el atributo onclick u otros eventos.
  • Interno: Dentro de la etiqueta <script> en el mismo archivo HTML.
  • Externo: En un archivo .js separado y enlazado con <script src="archivo.js"></script>.

Se recomienda escribir JavaScript en un archivo externo para mantener el código organizado y reutilizable.

Sintaxis básica de JavaScript

JavaScript tiene una sintaxis simple y flexible. Aquí algunos aspectos clave:

  • Sentencias: Cada instrucción suele terminar con un punto y coma (;), aunque no es obligatorio.
  • Comentarios: Se pueden escribir comentarios con // comentario en una línea o /* comentario en varias líneas */.
  • Case sensitive: JavaScript distingue entre mayúsculas y minúsculas. miVariable y mivariable son diferentes.
  • Indentación: se refiere a la técnica de agregar un espacio inicial (indentado) al principio de las líneas de código, que ayuda a delimitar visualmente los bloques y estructuras de control.

Respetar estas reglas es fundamental para escribir código limpio y funcional.

Tipos de datos en JavaScript

En JavaScript, los datos se clasifican en diferentes tipos según su naturaleza. Los principales tipos de datos son:

  • String (Cadenas de texto): Representan texto y se escriben entre comillas. Ejemplo: "Hola, mundo".
  • Number (Números): Representan valores numéricos, ya sean enteros o decimales. Ejemplo: 42, 3.14.
  • Boolean (Booleanos): Representan valores de verdadero (true) o falso (false).
  • Undefined: Una variable que ha sido declarada pero no tiene valor asignado.
  • Null: Representa la ausencia intencionada de un valor. Es un valor primitivo que indica la ausencia de valor.
  • Object (Objetos): Estructuras de datos que pueden contener múltiples valores en forma de pares clave-valor. Los objetos pueden almacenar cualquier tipo de dato.
  • Array (Arreglos): Un tipo especial de objeto que almacena una colección de valores. Los elementos de un arreglo se acceden mediante índices numéricos.
  • Symbol: Un tipo de dato primitivo introducido en ECMAScript 6. Los símbolos son valores únicos e inmutables que pueden ser utilizados como claves de propiedades de objetos, lo que ayuda a evitar colisiones de nombres entre propiedades de objetos.
  • BigInt: Un tipo de dato que permite representar números enteros que son demasiado grandes para ser representados como valores de tipo Number. Ejemplo: 9007199254740991n.
  • Function: Aunque no es un tipo de dato "primitivo", las funciones en JavaScript son objetos de primera clase y se pueden tratar como valores. Ejemplo: function miFuncion() {}.

JavaScript es un lenguaje de tipado dinámico, lo que significa que no es necesario declarar el tipo de una variable, ya que puede cambiar en tiempo de ejecución.

Variables en JavaScript

Una variable es un espacio en memoria donde se almacena un valor. En JavaScript, existen tres formas principales de declarar variables:

  • var: Forma tradicional de declarar variables, pero tiene un alcance menos controlado.
  • let: Se usa para declarar variables cuyo valor puede cambiar. Tiene un alcance limitado al bloque donde se declara.
  • const: Se usa para declarar constantes, es decir, valores que no cambiarán durante la ejecución del programa.

Es recomendable usar let y const en lugar de var para evitar errores y hacer el código más seguro.

Buenas prácticas para nombrar variables

  • Camel Case: Usar minúsculas para la primera palabra y mayúsculas para cada palabra subsecuente. Ejemplo: miVariable, nombreCompleto, cantidadDeArticulos.
  • Snake Case: Separar las palabras con guiones bajos. Ejemplo: nombre_completo, cantidad_de_articulos. Se usa generalmente para constantes o cuando se importa código de otros lenguajes.
  • Uso de Mayúsculas: Las constantes se escriben completamente en mayúsculas y se separan con guiones bajos. Ejemplo: PI, MAX_LONGITUD, NUMERO_DE_USUARIOS.
  • Uso del Símbolo de Dólar ($): El símbolo de dólar es válido en los nombres de variables y se utiliza frecuentemente en bibliotecas como jQuery para identificar elementos del DOM. Ejemplo: elemento$, miVariable$.
  • Nombres descriptivos: Es importante usar nombres de variables que sean descriptivos y representen claramente el valor que almacenan. Ejemplo: nombreUsuario, cantidadTotal, edadCliente.
  • Evitar caracteres especiales: No utilices caracteres especiales como !, @, %, etc. Solo se permiten letras, números, guiones bajos (_) y símbolos de dólar ($).
  • Empieza con una letra: Los nombres de las variables deben empezar con una letra, un guion bajo (_) o el símbolo de dólar ($). No deben comenzar con un número.
  • Evitar palabras reservadas: No utilices palabras reservadas de JavaScript como let, const, var, if, function, class, etc., como nombres de variables.

Siguiendo estas buenas prácticas, puedes escribir código más limpio, legible y mantenible.

Operadores en JavaScript

Los operadores permiten realizar operaciones con valores y variables en JavaScript. Se clasifican en diferentes tipos:

Operadores aritméticos

Se usan para realizar cálculos matemáticos.

  • + (Suma): 5 + 3 // 8
  • - (Resta): 10 - 4 // 6
  • * (Multiplicación): 6 * 2 // 12
  • / (División): 8 / 2 // 4
  • % (Módulo o residuo): 10 % 3 // 1
  • ** (Exponente): 2 ** 3 // 8

Operadores de asignación

Se usan para asignar valores a las variables.

  • = (Asignación): x = 10
  • += (Suma y asignación): x += 5 // x = x + 5
  • -= (Resta y asignación): x -= 2 // x = x - 2
  • *= (Multiplicación y asignación): x *= 3 // x = x * 3
  • /= (División y asignación): x /= 4 // x = x / 4

Operadores de comparación

Se usan para comparar valores y devuelven un resultado booleano (true o false).

  • == (Igualdad): 5 == "5" // true (compara solo el valor)
  • === (Igualdad estricta): 5 === "5" // false (compara valor y tipo de dato)
  • != (Diferente): 10 != 5 // true
  • !== (Diferente estricto): 10 !== "10" // true
  • > (Mayor que): 8 > 3 // true
  • < (Menor que): 4 < 7 // true
  • >= (Mayor o igual que): 6 >= 6 // true
  • <= (Menor o igual que): 3 <= 5 // true

Operadores lógicos

Se usan para combinar expresiones booleanas.

  • && (AND): true && false // false (ambos deben ser true para devolver true)
  • || (OR): true || false // true (al menos uno debe ser true)
  • ! (NOT): !true // false (niega el valor booleano)

Los operadores son esenciales para trabajar con condiciones y cálculos en JavaScript.

Strings en JavaScript

Un string es una cadena de caracteres utilizada para representar texto. Se puede definir usando comillas simples (' '), comillas dobles (" ") o template literals (` `).

Concatenación de Strings

La concatenación permite unir dos o más strings. Se puede hacer de dos formas:

  • Usando el operador +: "Hola" + " mundo": "Hola mundo"
  • Usando Template Literals: Se utilizan comillas invertidas (`) y la sintaxis ${} para insertar variables.

Métodos útiles para Strings

JavaScript proporciona varios métodos para manipular strings:

  • .length: Devuelve la longitud del string.
  • .toUpperCase(): Convierte el texto a mayúsculas.
  • .toLowerCase(): Convierte el texto a minúsculas.
  • .trim(): Elimina espacios en blanco al inicio y al final.
  • .includes("texto"): Verifica si una cadena contiene otra.
  • .slice(inicio, fin): Extrae parte de un string.
  • .replace("viejo", "nuevo"): Reemplaza parte de un string
  • .split("separador"): Divide un string en un array.

Estos métodos facilitan la manipulación de texto en JavaScript.

Números en JavaScript

En JavaScript, los números se representan con el tipo de dato Number. No hay diferencia entre enteros y decimales, ambos se manejan bajo el mismo tipo.

Operaciones con Números

Se pueden realizar operaciones matemáticas usando operadores aritméticos como + (suma), - (resta), * (multiplicación), / (división), % (módulo) y ** (exponente).

Métodos útiles para Números

JavaScript proporciona varios métodos para trabajar con números:

  • isInteger(numero): Verifica si un número es un entero.
  • parseInt(string): Convierte un string en un número entero.
  • parseFloat(string): Convierte un string en un número decimal.
  • toFixed(decimales): Redondea un número a la cantidad de decimales especificada.
  • toPrecision(dígitos): Formatea un número con una cantidad específica de dígitos significativos.
  • Math.round(numero): Redondea al entero más cercano.
  • Math.floor(numero): Redondea hacia abajo.
  • Math.ceil(numero): Redondea hacia arriba.
  • Math.random(): Genera un número decimal aleatorio entre 0 y 1.
  • Math.abs(numero): Devuelve el valor absoluto de un número.
  • Math.max(num1, num2, ...): Devuelve el número más grande de una lista.
  • Math.min(num1, num2, ...): Devuelve el número más pequeño de una lista.
  • Math.pow(base, exponente): Calcula una potencia.
  • Math.sqrt(numero): Calcula la raíz cuadrada de un número.

Estos métodos permiten realizar cálculos y manipular valores numéricos de forma sencilla en JavaScript.

(Type Casting) Conversión de Tipos de Datos

En JavaScript, es común tener que convertir valores de un tipo de dato a otro. Para ello, JavaScript ofrece varias funciones y métodos.

Conversión de String a Número

Si tenemos un valor de tipo string y queremos convertirlo a un número, podemos usar los siguientes métodos:

  • Number(valor): Convierte el valor a un número, si es posible. Si no puede convertirlo, devuelve NaN.
  • parseInt(valor): Convierte un string en un número entero. Si el string no comienza con un número, devuelve NaN.
  • parseFloat(valor): Convierte un string en un número decimal. Al igual que parseInt(), devuelve NaN si no puede realizar la conversión.

Conversión de Número a String

Para convertir un número a un string, se pueden usar los siguientes métodos:

  • toString(): Convierte un número a su representación en string. Ejemplo: 123.toString(): "123"
  • String(valor): Convierte cualquier tipo de valor (número, objeto, etc.) a string.

Conversión a Booleano

En JavaScript, ciertos valores se consideran "falsy" (falsos) y otros "truthy" (verdaderos). Para convertir cualquier valor a un booleano, puedes usar la función Boolean(valor), que devuelve:

  • Falsy: undefined, null, 0, NaN, "", false
  • Truthy: Cualquier valor que no sea falsy.

Ventanas de Diálogo en JavaScript

En JavaScript, se pueden utilizar ventanas de diálogo para interactuar con el usuario. Estas ventanas son proporcionadas por el navegador y permiten mostrar mensajes, recibir respuestas o confirmar acciones. Las principales ventanas de diálogo son alert(), confirm(), y prompt().

1. alert()

La función alert() muestra una ventana emergente con un mensaje y un botón "Aceptar". Esta ventana no devuelve ningún valor y simplemente sirve para mostrar información al usuario. Cuando aparece, detiene la ejecución del código hasta que el usuario haga clic en "Aceptar".

2. confirm()

La función confirm() muestra una ventana con un mensaje y dos botones: "Aceptar" y "Cancelar". Devuelve un valor booleano: true si el usuario hace clic en "Aceptar", y false si hace clic en "Cancelar". Este método se usa comúnmente para confirmar una acción o decisión del usuario, como la eliminación de un elemento.

3. prompt()

La función prompt() permite al usuario ingresar un valor en un campo de texto. Muestra un mensaje, un campo de entrada y dos botones: "Aceptar" y "Cancelar". El valor ingresado por el usuario se devuelve como un string. Si el usuario hace clic en "Cancelar", se devuelve null.

4. Otros tipos de ventanas emergentes (no estándar)

Algunos navegadores también ofrecen otras funciones para mostrar ventanas emergentes, aunque no son tan comunes como las anteriores. Estas incluyen:

  • console.log(): Aunque no es una ventana emergente, se utiliza para mostrar información en la consola del navegador, útil para depuración.
  • window.open(): Abre una nueva ventana del navegador con una URL específica. Es comúnmente usada para abrir páginas en nuevas pestañas o ventanas.

Impacto en el flujo de ejecución

Las funciones alert(), confirm(), y prompt() interrumpen el flujo de ejecución de la página. Cuando una de estas ventanas aparece, detiene el código JavaScript hasta que el usuario haya interactuado con la ventana, lo que puede resultar en una mala experiencia de usuario si se usan en exceso. En general, es recomendable utilizarlas con moderación y considerar alternativas como modales o notificaciones personalizadas cuando sea necesario.

Selección de Elementos del DOM

En JavaScript, se puede seleccionar elementos del DOM (Document Object Model) para poder interactuar con ellos. Existen varias funciones dentro del objeto document que permiten seleccionar elementos de la página web para luego modificarlos o extraer información. A continuación se describen las funciones más comunes.

1. document.getElementById()

La función getElementById() permite seleccionar un elemento del DOM usando su identificador único (id). Esta función devuelve un solo elemento, ya que id debe ser único dentro de la página.

2. document.getElementsByClassName()

La función getElementsByClassName() selecciona todos los elementos que tienen una clase específica. Esta función devuelve una colección de nodos (una lista de elementos) que coinciden con la clase dada. Es importante notar que devuelve una colección en vivo, lo que significa que si se agregan o eliminan elementos con esa clase, la colección se actualizará automáticamente.

3. document.getElementsByTagName()

La función getElementsByTagName() selecciona todos los elementos del DOM que coinciden con una etiqueta HTML específica. Al igual que getElementsByClassName(), devuelve una colección de nodos en vivo.

4. document.querySelector()

La función querySelector(), selecciona el primer elemento del DOM que coincide con un selector CSS especificado. A diferencia de getElementById(), que selecciona por ID, esta función permite usar selectores más flexibles, como clases (.clase), IDs (#id) o etiquetas (etiqueta). Devuelve un único nodo, incluso si hay múltiples coincidencias, y la colección no es en vivo.

5. document.querySelectorAll()

querySelectorAll() selecciona todos los elementos que coinciden con un selector CSS especificado, devolviendo una NodeList estática (no en vivo). Esto significa que, a diferencia de getElementsByClassName() o getElementsByTagName(), si el DOM cambia después de la selección, la lista no se actualizará automáticamente. Se pueden usar métodos como .forEach() para recorrer los elementos de la lista.

Impacto en el flujo

Las funciones como getElementById(), getElementsByClassName(), y getElementsByTagName() permiten manipular elementos de forma directa. Al seleccionar un elemento, se puede modificar su contenido, estilo, atributos, entre otros. Es fundamental entender cómo funcionan las colecciones de nodos devueltas (en vivo o no en vivo) para evitar problemas de actualización y manipulación incorrecta de los elementos.

Modificación del contenido en el DOM

El DOM (Document Object Model) permite manipular dinámicamente el contenido de una página web. JavaScript ofrece múltiples métodos para modificar el texto, insertar nuevos elementos y eliminar nodos dentro del documento HTML.

1. Modificar el texto de un elemento

Existen diferentes formas de cambiar el contenido textual de un elemento:

  • textContent: Modifica o devuelve el contenido de texto sin interpretar etiquetas HTML.
  • innerHTML: Modifica el contenido del elemento e interpreta etiquetas HTML.
  • innerText: Similar a textContent, pero respeta los estilos CSS como display: none.

2. Insertar nuevos elementos en el DOM

JavaScript permite agregar nuevos elementos dentro del DOM utilizando diferentes métodos:

  • appendChild(): Agrega un nuevo elemento dentro de otro como su hijo.
  • prepend(): Inserta un nuevo elemento como el primer hijo de un contenedor.
  • insertAdjacentHTML(): Inserta contenido HTML en distintas posiciones respecto a un elemento ("beforebegin", "afterbegin", "beforeend", "afterend").

3. Eliminar contenido o elementos del DOM

Para eliminar elementos del DOM, se pueden utilizar los siguientes métodos:

  • removeChild(): Elimina un elemento hijo dentro de otro.
  • remove(): Borra directamente un elemento del DOM.
  • replaceChild(): Reemplaza un nodo hijo por otro.

Estos métodos permiten modificar dinámicamente el contenido de una página web, ya sea cambiando texto, insertando elementos o eliminando contenido de manera eficiente.

Estructuras de Control: `if`, `if else` y `else if`

En JavaScript, la estructura de control if permite ejecutar un bloque de código solo si se cumple una condición específica. Si la condición no se cumple, se pueden usar else y else if para manejar otros casos posibles.

1. if: Evaluando una Condición

La sentencia if se usa para ejecutar código solo si una condición es verdadera.

Sintaxis:


if (condición) {
    // Código que se ejecuta si la condición es verdadera
}

2. if else: Manejo de Condiciones Alternativas

Cuando se necesita ejecutar un bloque de código en caso de que la condición no se cumpla, se usa else.

Sintaxis:


if (condición) {
    // Código si la condición es verdadera
} else {
    // Código si la condición es falsa
}

3. else if: Evaluando Múltiples Condiciones

Si se tienen varias condiciones posibles, se puede usar else if para evaluar diferentes escenarios antes de llegar al else.

Sintaxis:


if (condición1) {
    // Código si condición1 es verdadera
} else if (condición2) {
    // Código si condición2 es verdadera
} else {
    // Código si ninguna de las condiciones anteriores es verdadera
}

Consideraciones Importantes

  • Las condiciones dentro de if deben evaluarse a un valor true o false.
  • El bloque else es opcional, pero se recomienda para manejar casos no contemplados.
  • Se pueden anidar múltiples else if, pero es importante no abusar de ellos para evitar código difícil de leer.
  • Es recomendable utilizar operadores lógicos como && (AND), || (OR) y ! (NOT) para combinar condiciones dentro de un mismo if.

Las estructuras if, else if y else son fundamentales en la toma de decisiones dentro de un programa en JavaScript.
