Historia de Internet

Internet tiene sus orígenes en la década de 1960 con ARPANET, una red creada por el Departamento de Defensa de los Estados Unidos para la comunicación entre universidades y centros de investigación. Con el tiempo, evolucionó hasta convertirse en la red global que conocemos hoy.

El desarrollo del protocolo TCP/IP en la década de 1980 permitió la interconexión de redes, dando lugar al Internet moderno. A medida que crecía, surgieron nuevas tecnologías, como la web, que facilitaban el acceso a la información.

Historia de la Web

Aunque a menudo se confunden, Internet y la Web no son lo mismo. Internet es la infraestructura física y lógica que conecta dispositivos en todo el mundo (servidores, routers, cables, etc.), mientras que la Web es un servicio dentro de Internet que permite acceder a documentos interconectados mediante enlaces.

Principales hitos en la evolución de la Web:

  • 1989 - Inicio del desarrollo de la Web: Tim Berners-Lee, un científico del CERN, propuso un sistema para compartir información basado en hipertexto.
  • 1991 - Lanzamiento de la primera página web: La Web se basó en tres pilares fundamentales:
    • HTTP (Hypertext Transfer Protocol): Protocolo de comunicación que permite la transferencia de información.
    • HTML (Hypertext Markup Language): Lenguaje de marcado con el que se estructuran las páginas web.
    • URL (Uniform Resource Locator): Dirección única que identifica cada página web en la red.
  • 1992 - Nacimiento de HTML: HTML surgió a partir de SGML y fue desarrollado por Tim Berners-Lee para estructurar documentos en la web.
  • 1994 - Creación del W3C: Se fundó el World Wide Web Consortium (W3C) para definir los estándares de la web.
  • 1998 - HTML 4: Una de las versiones más duraderas de HTML, con mejoras en la estructura y formato de los documentos web.
  • 1999 - HTML 4.01: Una actualización menor de HTML 4 que, aunque corregía errores, fue considerada una de las peores versiones por su falta de innovación.
  • 2004 - Creación de WHATWG: Un grupo de desarrolladores y empresas, insatisfechos con la dirección de la W3C, fundaron el Web Hypertext Application Technology Working Group (WHATWG) y comenzaron a trabajar en HTML5.
  • 2008 - Lanzamiento de HTML5 por WHATWG: Se presentó como una alternativa más flexible y moderna para el desarrollo web.
  • 2014 - HTML5 se convierte en estándar oficial: La W3C adopta HTML5 como la nueva base para el desarrollo web.
  • 2017 - HTML5.2: Nueva actualización con mejoras en seguridad y accesibilidad.

Conceptos Fundamentales de HTML

¿Qué es HTML?

HTML (HyperText Markup Language) es un lenguaje de marcado utilizado para definir la estructura de las páginas web. A diferencia de los lenguajes de programación, HTML no ejecuta lógica o cálculos, sino que organiza y presenta el contenido de manera estructurada.

  • HTML no es un lenguaje de programación, sino un lenguaje de estructura.
  • No se encarga del diseño o los estilos (eso lo hace CSS), aunque se pueden aplicar estilos básicos.
  • Es la base sobre la cual están construidas todas las páginas web.
  • Cada etiqueta de HTML indica al navegador y a los motores de búsqueda la estructura y el significado del contenido.

Estructura de un Documento HTML

Un documento HTML está compuesto por etiquetas que estructuran la información. Cada etiqueta tiene una función específica y puede contener texto, imágenes, enlaces y otros elementos.

Reglas para Nombres de Archivos y Carpetas

  • Se recomienda escribir los nombres en minúsculas.
  • Para separar palabras, se utilizan guiones medios (-) en lugar de espacios o guiones bajos (_).
  • Ejemplo de nombre de carpeta: hola-mundo-web

Estructura Básica de HTML

Un documento HTML está formado por etiquetas, que pueden tener una etiqueta de apertura y su correspondiente etiqueta de cierre.

  • Etiqueta de apertura: Define el inicio de un elemento, por ejemplo, <p> para un párrafo.
  • Etiqueta de cierre: Indica el final del elemento, por ejemplo, </p>.
  • Ejemplo:
        <p>Este es un párrafo en HTML</p>

Comentarios en HTML

Los comentarios en HTML se utilizan para agregar notas dentro del código sin que sean visibles en la página web. Se escriben de la siguiente manera:

        <!-- Este es un comentario en HTML -->

Elementos de Texto

Encabezados

Los encabezados en HTML se utilizan para definir títulos y subtítulos dentro de una página web. Se dividen en seis niveles, desde <h1> (el más importante) hasta <h6> (el menos importante).

  • <h1> - Título principal (solo debería haber uno por página).
  • <h2> - Subtítulo o sección importante.
  • <h3> - Subsección dentro de un <h2>.
  • <h4>, <h5>, <h6> - Encabezados de menor importancia.

Párrafos

El elemento <p> se utiliza para definir párrafos de texto en HTML. Cada <p> crea un bloque de texto con espacio antes y después. 

        <p>Este es un párrafo de ejemplo en HTML.</p>

Elementos de Estructura Principal

<header> - Encabezado

Se usa para definir el encabezado de una página o de una sección. Generalmente contiene logotipos, títulos o menús de navegación.

    <header>
        <h1>Mi Sitio Web</h1>
    </header>

<nav> - Navegación

Define una sección de navegación que contiene enlaces a otras partes del sitio o a páginas externas.

    <nav>
        <ul>
            <li><a href="index.html">Inicio</a></li>
            <li><a href="blog.html">Blog</a></li>
            <li><a href="contacto.html">Contacto</a></li>
        </ul>
    </nav>

<main> - Contenido Principal

Contiene el contenido principal de la página. Cada página debe tener solo un <main>. 

    <main>
        <h2>Bienvenido a Nuestro Sitio</h2>
        <p>Aquí encontrarás información útil sobre diversos temas.</p>
    </main>

<footer> - Pie de Página

Define el pie de página de un documento o una sección. Generalmente contiene información de contacto, derechos de autor o enlaces adicionales.

    <footer>
        <p>© 2025 Mi Sitio Web - Todos los derechos reservados</p>
        <nav>
            <a href="politica-privacidad.html">Política de Privacidad</a> | 
            <a href="terminos.html">Términos y Condiciones</a>
        </nav>
    </footer>

Elementos de Sección o Contenedores

<section> - Sección

Se utiliza para agrupar contenido relacionado dentro de una página. Es útil cuando queremos dividir una página en secciones lógicas.

  <section>
      <h2>Noticias Recientes</h2>
      <p>Este es un artículo sobre las últimas noticias.</p>
  </section>

<article> - Artículo

Representa contenido independiente y reutilizable, como un artículo de blog, una noticia o un comentario. 

  <article>
      <h2>Título del Artículo</h2>
      <p>Este artículo puede leerse de forma independiente.</p>
  </article>

<aside> - Contenido Relacionado

Se usa para información complementaria, como barras laterales, enlaces relacionados o anuncios.

  <aside>
      <h3>Artículos Relacionados</h3>
      <ul>
          <li><a href="#">Cómo aprender HTML</a></li>
          <li><a href="#">Introducción a CSS</a></li>
      </ul>
  </aside>

<div> - División Genérica

Un contenedor sin semántica específica, útil para agrupar elementos y aplicar estilos con CSS.

  <div>
      <p>Este es un contenedor genérico.</p>
  </div>

<span> - Contenedor en Línea

Sirve para aplicar estilos o agrupar elementos dentro de un texto sin romper la estructura del contenido. 

  <p>Esto es un <span>texto resaltado</span> dentro de un párrafo.</p>

Elementos de Formato y Estilo de Texto

  • <em> - Énfasis: Texto en énfasis.
  • <strong> - Énfasis fuerte: Texto importante.
  • <small> - Texto más pequeño: Texto reducido.
  • <br> - Salto de línea (sin etiqueta de cierre).
  • <wbr> - Salto de línea si es necesario.
  • <time> - Representación de fechas y horas: 24 de febrero de 2025
  • <i> - Texto en cursiva: Ejemplo de texto en cursiva.
  • <b> - Texto en negrita: Ejemplo de texto en negrita.
  • <u> - Texto subrayado: Ejemplo de texto subrayado.
  • <sup> - Superíndice (elevado): 102 = 100
  • <sub> - Subíndice (bajo): H2O

Atributos en HTML

Los atributos en HTML son valores adicionales que se agregan a las etiquetas para configurar su comportamiento o apariencia.

Atributos comunes:

  • class: Se usa para aplicar estilos CSS a un grupo de elementos con la misma clase.
  • id: Identificador único de un elemento, utilizado para manipulación con JavaScript y navegación entre anclas.
  • title: Muestra un mensaje emergente (tooltip) cuando el usuario pasa el cursor sobre el elemento.
  • data-*: Permite almacenar datos personalizados dentro de un elemento HTML.

Ejemplo:

<p class="resaltado" id="parrafo1" title="Esto es un párrafo" data-info="ejemplo">
    Este es un párrafo con varios atributos.
</p>

Enlaces en HTML

Los enlaces (<a>) permiten conectar una página web con otra, enlazar recursos internos o externos y crear navegación entre secciones.

Atributos de los enlaces:

  • href: Define la ruta o URL del recurso al que queremos acceder.
  • target: Indica dónde se abrirá el enlace. Los valores más usados son:
    • _self (predeterminado) - Abre el enlace en la misma pestaña.
    • _blank - Abre el enlace en una nueva pestaña o ventana.
  • download: Permite descargar un archivo en lugar de abrirlo en el navegador. Solo funciona con archivos en el mismo servidor.

Ejemplo:

<a href="https://www.ejemplo.com" target="_blank" title="Ir a Ejemplo">
    Visitar Ejemplo
</a>

Anclas en HTML

Las anclas permiten la navegación dentro de la misma página. Para ello, usamos el atributo id en un elemento destino y un enlace con # seguido del id.

Ejemplo:

<a href="#seccion2">Ir a la Sección 2</a>

<h2 id="seccion2">Sección 2</h2>

Rutas en HTML

Existen dos tipos de rutas para acceder a recursos:

1. Rutas absolutas

Son direcciones completas con protocolo (https://) que apuntan a un recurso en internet. Se usan para enlaces externos.

<a href="https://www.google.com">Ir a Google</a>

2. Rutas relativas

Apuntan a archivos dentro del mismo proyecto sin especificar el dominio.

  • Mismo nivel: Si el archivo está en la misma carpeta, solo se pone el nombre del archivo. 
    <a href="pagina.html">Abrir Página</a>
  • Subcarpeta: Si está dentro de otra carpeta, se indica la ruta. 
    <a href="carpeta/pagina.html">Abrir Página en Carpeta</a>
  • Salir de la carpeta: Se usa ../ para retroceder un nivel en la estructura de carpetas. 
    <a href="../pagina.html">Volver a Página Principal</a>

Más Etiquetas en HTML

Elementos de contenido estructural:

  • <address>: Se utiliza para representar información de contacto, como direcciones físicas o correos electrónicos.
  • <blockquote>: Sirve para marcar citas de otros autores, destacando el contenido citado.
  • <pre>: Muestra texto preformateado, respetando los espacios y saltos de línea.
  • <code>: Se usa para representar fragmentos de código de programación.
  • <div>: Es un contenedor genérico que se usa para estructurar el contenido, pero no tiene un significado semántico.
  • <hr>: Inserta una línea horizontal que separa contenido en la página.

Ejemplo:

  <address>
      Contacto: contacto@ejemplo.com
  </address>
  
  <blockquote>
      "La mejor forma de predecir el futuro es crearlo." - Peter Drucker
  </blockquote>
  
  <pre>
      function holaMundo() {
          console.log("¡Hola, mundo!");
      }
  </pre>
  
  <code>console.log("Hola, mundo");</code>
  
  <hr>

Listas en HTML

Tipos de listas:

  • Listas desordenadas (<ul>): No tienen un orden específico y se representan con viñetas.
  • Listas ordenadas (<ol>): Tienen un orden numérico o con letras.
  • Listas de definición (<dl>): Se usan para definir términos y sus descripciones.
  • Listas anidadas: Una lista dentro de otra lista, útil para subcategorías.

Ejemplo de listas en HTML:

Lista desordenada:

  <ul>
      <li>Manzana</li>
      <li>Banana</li>
      <li>Uva</li>
  </ul>

Lista ordenada:

  <ol>
      <li>Primer paso</li>
      <li>Segundo paso</li>
      <li>Tercer paso</li>
  </ol>

Lista de definición:

  <dl>
      <dt>HTML</dt>
      <dd>Lenguaje de marcado para la creación de páginas web.</dd>
  
      <dt>CSS</dt>
      <dd>Lenguaje para definir estilos en documentos HTML.</dd>
  </dl>

Lista anidada:

  <ul>
      <li>Frutas
          <ul>
              <li>Manzana</li>
              <li>Naranja</li>
          </ul>
      </li>
      <li>Verduras
          <ul>
              <li>Lechuga</li>
              <li>Zanahoria</li>
          </ul>
      </li>
  </ul>

Imágenes en HTML

<img> - Insertar una Imagen

La etiqueta <img> se usa para mostrar imágenes en una página web. No tiene etiqueta de cierre.

    <img src="imagen.jpg" alt="Descripción de la imagen">

Atributos importantes:

  • src: Define la ruta de la imagen.
  • alt: Proporciona una descripción para accesibilidad y en caso de que la imagen no cargue.
  • width y height: Permiten definir el tamaño de la imagen (en píxeles o porcentaje).

Ejemplo de Imagen con Dimensiones Definidas

    <img src="paisaje.jpg" alt="Paisaje hermoso" width="400" height="250">

Imágenes desde una URL Externa

    <img src="https://www.ejemplo.com/imagen.jpg" alt="Imagen desde Internet">

Uso de <figure> y <figcaption>

Para agregar una descripción semántica a la imagen, se puede usar <figure> y <figcaption>. 

    <figure>
        <img src="foto.jpg" alt="Persona programando">
        <figcaption>Imagen de una persona programando.</figcaption>
    </figure>

Audio y Video en HTML

<audio> - Insertar Audio

La etiqueta <audio> permite incluir archivos de audio en una página web.

            <audio controls>
                <source src="audio.mp3" type="audio/mpeg">
                Tu navegador no soporta el elemento de audio.
            </audio>

Atributos importantes:

  • controls: Muestra controles de reproducción (play, pausa, volumen).
  • autoplay: Hace que el audio se reproduzca automáticamente.
  • loop: Hace que el audio se repita en bucle.

<video> - Insertar Video

La etiqueta <video> permite mostrar videos en la página web.

            <video width="640" height="360" controls>
                <source src="video.mp4" type="video/mp4">
                Tu navegador no soporta el elemento de video.
            </video>

Atributos importantes:

  • controls: Muestra los controles de reproducción.
  • width y height: Definen el tamaño del video.
  • poster: Imagen que se muestra antes de reproducir el video.
  • autoplay: Reproduce el video automáticamente.
  • loop: Hace que el video se repita en bucle.

Ejemplo de Video con Imagen de Vista Previa

            <video width="640" height="360" controls poster="preview.jpg">
                <source src="video.mp4" type="video/mp4">
                Tu navegador no soporta el elemento de video.
            </video>
