Historia de CSS

CSS (Cascading Style Sheets - Hojas de Estilo en Cascada) es el lenguaje utilizado para definir la presentación y el diseño de las páginas web. Su historia está ligada a la evolución del desarrollo web y la necesidad de separar el contenido (HTML) del diseño.

Principales hitos en la evolución de CSS:

  • 1994 - Nacimiento de CSS: Håkon Wium Lie propuso el concepto de hojas de estilo en cascada mientras trabajaba en el CERN junto a Tim Berners-Lee.
  • 1996 - CSS1: El W3C publicó la primera versión oficial de CSS, que permitía definir colores, fuentes y alineaciones básicas.
  • 1998 - CSS2: introdujeron nuevas características, como posicionamiento absoluto, z-index y estilos para medios de impresión.
  • 1999-2000 - Falta de soporte en navegadores: Durante varios años, los navegadores implementaron CSS de manera inconsistente, lo que dificultó su adopción.
  • 2004 - CSS3: Se propuso dividir CSS en módulos para facilitar su evolución, agregando características avanzadas como animaciones, sombras y transiciones.
  • 2011 - Estándar CSS3: Los navegadores comenzaron a implementar CSS3 de manera más uniforme, convirtiéndolo en la base del diseño web moderno.
  • Actualidad - CSS4 (en desarrollo): Aunque no existe un "CSS4" oficial, el desarrollo de nuevas funcionalidades sigue en marcha con la evolución de los módulos CSS.

Conceptos Fundamentales de CSS

¿Qué es CSS?

CSS (Cascading Style Sheets) es un lenguaje de estilos utilizado para controlar la apariencia y el diseño de las páginas web. Se usa junto con HTML para definir colores, fuentes, márgenes, posiciones y mucho más.

  • CSS permite separar el contenido (HTML) de la presentación (diseño y estilos).
  • Facilita la creación de diseños responsivos y reutilizables.
  • Funciona siguiendo la cascada, donde los estilos más específicos tienen mayor prioridad.

Reglas para Nombres de Archivos CSS

  • Se recomienda escribir los nombres en minúsculas.
  • Para separar palabras, se utilizan guiones medios (-) en lugar de espacios o guiones bajos (_).
  • Ejemplo de nombre de archivo: estilos-principales.css

¿Cómo Agregar CSS a un Documento HTML?

Existen tres formas principales de aplicar CSS en una página web:

  • CSS en línea: Se usa el atributo style dentro de una etiqueta HTML.
  • CSS interno: Se define dentro de la etiqueta <style> en el <head> del documento.
  • CSS externo: Se enlaza un archivo .css mediante la etiqueta <link>.

Ejemplo de CSS en un archivo externo:

            
            <link rel="stylesheet" href="estilos.css">

            /* Archivo: estilos.css */
            body {
            background-color: #f0f0f0;
            font-family: Arial, sans-serif;
            }

Comentarios en CSS

Los comentarios en CSS permiten agregar notas dentro del código sin afectar el diseño. Se escriben de la siguiente manera:

        /* Este es un comentario en CSS */

Propiedad background en CSS

La propiedad background en CSS se utiliza para definir el fondo de un elemento. Puede incluir colores, imágenes, gradientes y más.

Principales propiedades de background

  • background-color: Define un color de fondo.
  • background-image: Establece una imagen de fondo.
  • background-repeat: Controla la repetición de la imagen de fondo.
  • background-position: Define la posición de la imagen de fondo.
  • background-size: Controla el tamaño de la imagen de fondo.
  • background-attachment: Determina si la imagen de fondo se desplaza con la página o es fija.
  • background: Propiedad abreviada para definir varias propiedades en una sola línea.

Ejemplo de uso

A continuación, un ejemplo de cómo aplicar diferentes fondos en CSS:

        /* Fondo de color sólido */
        body {
            background-color: lightblue;
        }
        
        /* Fondo con imagen */
        .container {
            background-image: url('imagen.jpg');
            background-size: cover;
            background-position: center;
            background-repeat: no-repeat;
        }
        
        /* Fondo con gradiente */
        .gradient-box {
            background: linear-gradient(to right, blue, pink);
        }

Valores comunes para background-size

  • auto: Tamaño original de la imagen.
  • cover: Ajusta la imagen para cubrir todo el elemento sin distorsión.
  • contain: Ajusta la imagen para que quepa dentro del elemento sin recortarla.

Ejemplo en HTML

        <div class="container">
            <p>Este div tiene una imagen de fondo</p>
        </div>

Estilización de Texto en CSS

¿Cómo se maneja el texto en CSS?

CSS ofrece varias propiedades para modificar el aspecto del texto en una página web. Se pueden cambiar aspectos como el color, tamaño, espaciado, alineación y fuente.

Principales propiedades para texto

  • color: Define el color del texto.
  • font-size: Establece el tamaño de la fuente.
  • font-family: Define la fuente del texto.
  • font-weight: Controla el grosor del texto.
  • text-align: Alinea el texto (izquierda, derecha, centro o justificado).
  • text-decoration: Agrega subrayados, tachados o elimina decoraciones.
  • text-transform: Convierte el texto a mayúsculas, minúsculas o capitaliza cada palabra.
  • letter-spacing: Ajusta el espaciado entre caracteres.
  • line-height: Define el espaciado entre líneas de texto.

Ejemplo de uso

A continuación, un ejemplo de cómo aplicar estilos al texto en CSS:

            /* Texto con color y tamaño */
            h1 {
                color: darkblue;
                font-size: 32px;
                font-family: Arial, sans-serif;
            }
            
            /* Alinear y modificar el peso del texto */
            p {
                text-align: justify;
                font-weight: bold;
            }
            
            /* Convertir texto a mayúsculas */
            .uppercase {
                text-transform: uppercase;
            }
            
            /* Subrayar texto */
            .underline {
                text-decoration: underline;
            }

Ejemplo en HTML

        <h1>Título Importante</h1>
        <p class="uppercase">Este es un texto en mayúsculas.</p>
        <p class="underline">Este texto está subrayado.</p>

Márgenes en CSS

Los márgenes en CSS definen el espacio exterior alrededor de un elemento, separándolo de otros elementos en la página.

Propiedades de los márgenes

  • margin: Define el margen en los cuatro lados del elemento.
  • margin-top: Establece el margen superior.
  • margin-right: Define el margen derecho.
  • margin-bottom: Ajusta el margen inferior.
  • margin-left: Especifica el margen izquierdo.
  • margin: auto: Se usa para centrar un elemento horizontalmente.

Sintaxis y valores

Los márgenes pueden recibir diferentes valores:

  • Píxeles (px): margin: 20px;
  • Porcentajes (%): Relativo al ancho del contenedor padre (margin: 5%;).
  • Valores negativos: Reducen el espacio alrededor del elemento (margin: -10px;).
  • Shorthand (Atajo): Se pueden definir todos los márgenes en una sola línea:
        /* Todos los lados con el mismo margen */
        margin: 10px;
        
        /* Margen superior e inferior de 10px, izquierdo y derecho de 20px */
        margin: 10px 20px;
        
        /* Superior 10px, horizontal 20px, inferior 30px */
        margin: 10px 20px 30px;
        
        /* Superior 10px, derecho 15px, inferior 20px, izquierdo 25px */
        margin: 10px 15px 20px 25px;

Ejemplo en CSS

        /* Agregar margen a un párrafo */
        p {
            margin: 20px;
        }
        
        /* Margen automático para centrar un contenedor */
        .container {
            width: 80%;
            margin: auto;
        }

Ejemplo en HTML

        <div class="container">
            <p>Este es un párrafo con margen.</p>
        </div>

Padding en CSS

El padding (o relleno) en CSS define el espacio interno entre el contenido de un elemento y su borde. A diferencia del margen, que separa un elemento de otros, el padding afecta el espacio dentro del propio elemento.

Propiedades del padding

  • padding: Define el relleno en los cuatro lados del elemento.
  • padding-top: Establece el relleno superior.
  • padding-right: Define el relleno derecho.
  • padding-bottom: Ajusta el relleno inferior.
  • padding-left: Especifica el relleno izquierdo.

Sintaxis y valores

El padding puede definirse con diferentes valores:

  • Píxeles (px): padding: 15px;
  • Porcentajes (%): Relativo al ancho del contenedor padre (padding: 5%;).
  • Shorthand (Atajo): Se pueden definir todos los paddings en una sola línea:
        /* Padding igual en todos los lados */
        padding: 15px;
        
        /* Superior e inferior 10px, izquierdo y derecho 20px */
        padding: 10px 20px;
        
        /* Superior 10px, horizontal 20px, inferior 30px */
        padding: 10px 20px 30px;
        
        /* Superior 10px, derecho 15px, inferior 20px, izquierdo 25px */
        padding: 10px 15px 20px 25px;

Ejemplo en CSS

        /* Agregar relleno a un párrafo */
        p {
            padding: 20px;
            background-color: lightgray;
        }
        
        /* Relleno diferente en cada lado */
        .box {
            padding: 10px 20px 30px 40px;
            background-color: lightblue;
        }

Ejemplo en HTML

        <div class="box">
            <p>Este es un párrafo con padding.</p>
        </div>

Display: Block e Inline en CSS

¿Qué es la propiedad display?

La propiedad display en CSS define cómo un elemento debe comportarse en el flujo del documento. Determina si un elemento se muestra en línea, en bloque o en otro formato.

Display: Block

Los elementos con display: block ocupan todo el ancho disponible de su contenedor y generan un salto de línea antes y después de ellos.

Ejemplos de elementos de bloque:

  • <div>
  • <p>
  • <h1> a <h6>
  • <section>
  • <article>

Display: Inline

Los elementos con display: inline solo ocupan el ancho de su contenido y no generan saltos de línea. Pueden estar en la misma línea que otros elementos.

Ejemplos de elementos en línea:

  • <span>
  • <a> (enlaces)
  • <strong>
  • <em>

Ejemplo en CSS:

            /* Convertir un bloque en un elemento en línea */
            div {
                display: inline;
                background-color: lightblue;
            }

Selectores Avanzados en CSS

Selectores de Atributo

Los selectores de atributo permiten aplicar estilos a elementos en función de sus atributos y valores.

Ejemplos de uso:

  • [atributo]: Selecciona elementos que tienen el atributo especificado.
  • [atributo="valor"]: Selecciona elementos con un atributo específico y un valor exacto.
  • [atributo~="valor"]: Selecciona elementos cuyo atributo contiene una palabra específica.
  • [atributo^="valor"]: Selecciona elementos cuyo atributo comienza con un valor específico.
  • [atributo$="valor"]: Selecciona elementos cuyo atributo termina con un valor específico.
  • [atributo*="valor"]: Selecciona elementos cuyo atributo contiene una cadena específica.

Ejemplo en CSS:

            /* Seleccionar todos los enlaces que abren en una nueva pestaña */
            a[target="_blank"] {
                color: red;
                font-weight: bold;
            }
        
            /* Seleccionar inputs de tipo texto */
            input[type="text"] {
                border: 2px solid blue;
            }
        
            /* Seleccionar imágenes cuyo atributo src contenga "logo" */
            img[src*="logo"] {
                width: 150px;
            }

Pseudoclase :hover

La pseudoclase :hover se aplica cuando el usuario coloca el cursor sobre un elemento.

Ejemplo en CSS:

            /* Cambiar el color de un enlace al pasar el cursor */
            a:hover {
                color: green;
                text-decoration: underline;
            }
        
            /* Cambiar el fondo de un botón al pasar el cursor */
            button:hover {
                background-color: darkblue;
                color: white;
            }

Pseudoclase :nth-child()

La pseudoclase :nth-child(n) selecciona elementos en función de su posición dentro de su padre.

Ejemplos de uso:

  • :nth-child(even): Selecciona los elementos pares.
  • :nth-child(odd): Selecciona los elementos impares.
  • :nth-child(3): Selecciona el tercer elemento.
  • :nth-child(2n): Selecciona los elementos en posiciones múltiplos de 2.

Ejemplo en CSS:

            /* Colorear filas pares de una lista */
            ul li:nth-child(even) {
                background-color: lightgray;
            }
        
            /* Resaltar el tercer párrafo */
            p:nth-child(3) {
                font-weight: bold;
                color: red;
            }
