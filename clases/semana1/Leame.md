============================== DESARROLLO WEB FULL STACK ==============================
==========================================================================================

Un desarrollador/a web full stack es un profesional con habilidades en el desarrollo tanto del **frontend** (parte visual e interactiva de un sitio web) como del **backend** (lógica, base de datos y servidores).

📌 **Full Stack significa:**
✔ Saber estructurar páginas con **HTML**
✔ Diseñar interfaces atractivas con **CSS**
✔ Crear interactividad con **JavaScript**
✔ Construir y gestionar servidores con **Node.js, Python u otro lenguaje**
✔ Trabajar con bases de datos y APIs  **SQL, NoSQL**

🎯 Un full stack developer es capaz de desarrollar una aplicación web completa, desde la interfaz hasta la base de datos y su despliegue en producción.

==========================================================================================
============================== VOCABULARIO WEB ==============================
==========================================================================================

- **IP (Internet Protocol)**: Es un identificador numérico único asignado a un dispositivo en una red. En el contexto web, representa la dirección del servidor donde está alojado un sitio web.

- **Dominio web / URL (Uniform Resource Locator)**: Es el nombre asociado a una IP que facilita el acceso a un sitio web. Ejemplo: Google tiene la IP 216.58.210.163, pero accedemos a él mediante "google.com".

- **DNS (Domain Name System)**: Sistema que traduce los nombres de dominio en direcciones IP para que los usuarios no tengan que recordar números.

- **HTTP (Hypertext Transfer Protocol)**: Protocolo de comunicación que permite la transferencia de información en la web. No cifra los datos, por lo que no es seguro para información sensible.

- **HTTPS (Hypertext Transfer Protocol Secure)**: Versión segura de HTTP que utiliza cifrado SSL/TLS para proteger la información transmitida entre el navegador y el servidor, asegurando mayor privacidad y seguridad.

- **Servidor**: Ordenador o sistema que almacena y distribuye recursos web a los usuarios a través de la red.

- **Hosting**: Servicio que proporciona almacenamiento en servidores para que un sitio web esté accesible en Internet.

- **Petición**: Solicitud de recursos (archivos, imágenes, datos) que un navegador realiza a un servidor para mostrar una página web.

- **Sitio web**: Conjunto de páginas web y recursos (imágenes, videos, archivos, etc.) alojados en un mismo dominio.

- **SEO (Search Engine Optimization)**: Conjunto de estrategias y técnicas para mejorar la visibilidad y posicionamiento de un sitio web en motores de búsqueda como Google.
